# Fiche projet – Site de vente de bijoux

## Description
Projet de développement d’un site e-commerce de bijoux permettant la consultation de produits, la gestion d’un panier et le paiement en ligne.

## Technologies utilisées
- Frontend : HTML, CSS, JavaScript
- Backend : PHP
- Base de données : MySQL
- Paiement : Stripe
- Automatisation : Make (RSS Raindrop)
- Outils IA : NotebookLM (Gemini)

## Fonctionnalités principales
- Catalogue de bijoux
- Page détail produit
- Panier
- Paiement sécurisé
- Gestion du stock
- Administration (gestion des produits)

## Objectif
Créer une application web fonctionnelle et sécurisée permettant de simuler un site e-commerce réel.
