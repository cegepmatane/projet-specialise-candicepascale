# Analyse concurrentielle

## Concurrents principaux
- Shopify (plateforme e-commerce)
- Etsy (vente de bijoux artisanaux)
- Amazon (marketplace généraliste)

## Différences
- Projet personnalisé (pas une solution clé en main)
- Apprentissage technique (PHP, MySQL, API Stripe)
- Contrôle total sur les fonctionnalités

## Avantages
- Flexibilité du développement
- Adaptation aux besoins spécifiques
- Compréhension complète du système

## Limites
- Moins optimisé que les solutions professionnelles
- Moins scalable à grande échelle
